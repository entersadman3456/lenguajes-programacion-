Introducción

En la actualidad los lenguajes de programación han tomado un papel muy importante desarrollando sistemas o software, con el objetivo principal de facilitar al usuario las actividades que realiza día con día; por tal motivo, como programador, es importante conocer los conceptos básicos de programación, los tipos de lenguajes que se utilizan para el desarrollo y su funcionamiento para la interpretación de algoritmos, así como para dar solución a los problemas que pudieran presentarse. Existen muchos lenguagues de programación y cada uno funciona para cosas diferentes, como desarrollo y diseño web (HTML y JavaScript) y desarrollo de software (C y C++). Hay muchísimos más y cada uno tiene un alcance y forma de comunicación distinta. Permiten desde construir un sitio web y hacerlo funcionar desde cero, hasta desarrollar una aplicación para Android o iOS. .
